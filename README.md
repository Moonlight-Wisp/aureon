# Aureon - Next-Gen Operations Platform

Aureon is a premium, high-performance web experience showcasing modern front-end architecture, interactive 3D visualizations, and robust UI/UX design.

## 🚀 Features

- **"Wow Factor" Interactivity:**
  - Integrated **Three.js / WebGL** for a dynamic, particle-based hero background.
  - **GSAP ScrollTrigger** for cinematic scroll reveals, parallax, and staggered entrances.
  - Custom magnetic cursor with state-based hover effects.
  - Advanced 3D tilt effects on component cards.
- **Performance & Architecture:**
  - Built with **Vite** for fast HMR and optimized production builds.
  - Modular CSS Architecture (separated variables, components, and layout files) ensuring maintainability.
  - Modular ES6 JavaScript architecture.
  - 100/100/100/100 **Lighthouse Score** capability.
- **Advanced UX:**
  - ROI Simulator powered by **Chart.js**.
  - Mocked Chatbot Assistant mimicking API behavior with typing indicators.
  - Touch-swipe support on mobile carousels.

## 📦 Tech Stack

- **Vanilla HTML/CSS/JS**
- **Vite** (Bundler)
- **Three.js** (WebGL Interactions)
- **GSAP** (Scroll & UI Animations)
- **Chart.js** (Data Visualization)

## 💻 Running Locally

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
